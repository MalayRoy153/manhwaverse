# ManhwaVerse

A scalable, dark/light mode manhwa reading website.

## Live Site
https://malayroy153.github.io/manhwaverse/

## Features
- Dark / Light mode toggle
- Responsive reader with keyboard navigation (← →)
- Auto-hiding UI for immersive reading
- Easy chapter expansion via `js/data.js`

## How to Add New Chapters
1. Upload new board images to `assets/manhwa/[series]/chapter-X/`
2. Add the chapter entry to `js/data.js`
3. Commit the changes — the site updates automatically.
